# 💻 Front-end Desafio Full

### 📋 Sobre o Projeto
Este repositório contém a resolução de um desafio focado em desenvolvimento web front-end. 
O objetivo é demonstrar competências em estruturação de interface, estilização e interatividade, funcionando como um portfólio de habilidades técnicas.

### ✨ Funcionalidades
Interface responsiva para diferentes dispositivos.
Organização de componentes para facilitar a manutenção do código.
Implementação de elementos visuais dinâmicos e modernos.

### 🛠️ Tecnologias Utilizadas
**HTML5**: Estruturação semântica do conteúdo.
**CSS3**: Estilização, layout e design visual.
**JavaScript**: Lógica de programação e comportamento da interface.

### 🚀 Como Executar
Para rodar este projeto localmente no seu computador, siga estes passos baseados no fluxo do Git:

1. **Clonar o repositório**: Baixe os arquivos do GitHub para a sua máquina.
    ```bash
    git clone (https://github.com/abeatrizz/front-endDesafioFull.git)
    ```
2.  **Acessar a pasta**: Entre no diretório do projeto via terminal.
    ```bash
    cd front-endDesafioFull
    ```
3.  **Abrir o projeto**: Como este é um projeto de front-end, basta abrir o arquivo `index.html` em qualquer navegador.
